## Getting Started

1. Clone this repository to your local machine:
    ```shell
    git clone https://github.com/AsjidAle/DSA.git
    ```
2. Change to the project directory:
    ```shell
    cd DSA
    ```
3. You're there Now go to file your choice, compile and execute
